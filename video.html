Introduction

So you want to add a video to your website. Well This will show you how to do it.

Step 1: Find a video

For this tutorial we're going to grab a video from YouTube so go find one.

Step 2: Get the HTML code

Once we have found our video (I will be using my own video to avoid copyright
issues but you should get permission from the copyright holder on if you can
use their video). We will want to go to the share button.


Once on the share screen we want to select embed.

Then we want to copy the HTML code it gives us.

Step 3: Put it in your website

Now that we have our HTML code we need to put it into our website
so open up whatever text editor you are using for your operating system.
For this example I am using Visual Studio Code on Arch Linux.

Now you just paste the HTML code Youtube gave you in.


And now you have a YouTube video in your website.
